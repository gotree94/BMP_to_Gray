좌표 매핑 상세 분석

위의 버튼을 클릭하여 각 위치에서의 좌표 매핑을 확인하세요.

getPixel 함수 동작 원리

unsigned char getPixel(unsigned char* image, int width, int height, int x, int y) { // 좌표 클램핑 (범위 제한) if (x < 0) x = 0; // 왼쪽 경계 벗어남 → 0으로 설정 if (x >= width) x = width - 1; // 오른쪽 경계 벗어남 → width-1로 설정 if (y < 0) y = 0; // 위쪽 경계 벗어남 → 0으로 설정 if (y >= height) y = height - 1; // 아래쪽 경계 벗어남 → height-1로 설정 return image[y * width + x]; // 클램핑된 좌표로 픽셀 값 반환 }

핵심 포인트:

  • 데이터 복사나 보간 없음 - 기존 픽셀을 재사용
  • 경계 픽셀의 값이 여러 번 사용됨 - 가장자리에서 같은 값이 반복
  • 간단하고 효율적 - 추가 메모리나 복잡한 계산 불필요
  • 엣지 검출에 적합 - 경계에서 인위적인 엣지 생성 방지
