4. 경계 처리 (Boundary Handling)

이미지 가장자리 픽셀을 클릭하면 경계 처리 방법을 확인할 수 있습니다.

getPixel 함수의 클램핑:
if (x < 0) x = 0;
if (x >= width) x = width - 1;
if (y < 0) y = 0;
if (y >= height) y = height - 1;
